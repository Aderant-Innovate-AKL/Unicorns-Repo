Vector Similarity Search

Semantic Understanding with OpenSearch k-NN

üîç How It Works

  1. Convert name to 768-dimensional vector
  2. Use HNSW algorithm for nearest neighbors
  3. Cosine similarity scoring
  4. Return top-k candidates

üí° Why Vectors?

"John Smith" and "Jon Smyth" are semantically close in vector space‚Äîeven though they differ in spelling.

üìä Score Interpretation

Score Meaning
0.95-1.00 Nearly identical
0.85-0.94 Very similar
0.70-0.84 Moderately similar
<0.70 Weak match

‚ö° Performance

~50ms search time for 350K+ vectors
